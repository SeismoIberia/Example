## Reglas de publicación
- Si hay algún error en la nueva versión que aún no esté listado, por favor, envía la información del fallo a ```issue
```.
- ¡Si utilizas cualquier tipo de lenguaje insultante o despectivo hacia otros usuarios, serás baneado permanentemente!
- ¡No envíes contenido repetitivo o sin sentido a ```issue```, de lo contrario serás baneado permanentemente!
- Si tienes cualquier duda o sugerencia, eres bienvenido a plantearla.

## Colaboración
